Категории:
    HTTP GET по адресу:
    https://prompower.ru/api/getCategoryGroups
    ключ не нужен

Группы:
    используйте HTTP GET по адресу:
    https://prompower.ru/api/categories
    ключ не нужен

Товары:
используйте HTTP-POST https://prompower.ru/api/prod/getProducts
В заголовках запроса необходимо указать "Content-type": "application/json"
В теле запроса укажите email (указанный при регистрации) и ваш ключ:
{
"email": "",
"key": ""
}  

Документы на группу товаров:
    Используйте HTTP GET запрос по адресу:
    https://prompower.ru//api/docfiles?dir=inverters&subdir=pd310&filenameFilter=
    параметры:
    filenameFilter: обязательный, может быть пустой, либо строковое выражение. В случае не пустого значения будет произведен отбор по вхождению строки в имя файла
    dir: параметр slug из данных категории
    subdir: параметр slug из данных подкатегории

    На данный момент doctype бывает: doc, soft, text и other - применяйте на свое усмотрение

    для опциональных плат инверторов также есть отдельная документация на каждую плату. В этом случае к параметрам, помимо dir и subdir необходимо добавить еще и конкретный продукт. Параметр product и в качестве его значения строка, содержащая параметр article из данных продукта
